## 1. Project Goal

A **single app** where:
- Users **sign up/login with email**
- **Phone number is asked only during profile setup**
- Users can **donate**, **request blood**, **chat**, **become blood banks**
- Blood banks have **separate dashboard**
- Real-time chat, notifications, stock

## 9. Key Code Snippets

### **Email Signup (app/(auth)/signup.tsx)**

```tsx
const handleSignup = async () => {
  try {
    const userCred = await createUserWithEmailAndPassword(auth, email, password);
    await setDoc(doc(db, 'users', userCred.user.uid), {
      email,
      role: 'user',
      isBloodBank: false,
      profileComplete: false
    });
    router.replace('/profile-setup');
  } catch (e) { Alert.alert('Error', e.message); }
};
```

---

### **Switch to Blood Bank Mode**

